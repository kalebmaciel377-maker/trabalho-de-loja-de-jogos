Minecraft Java & Badrook

O Minecraft possui duas versões principais: Java Edition e Bedrock Edition. A versão Java é exclusiva para PC e permite o uso de mods e maior personalização. Já a Bedrock funciona em várias plataformas e possibilita que jogadores de diferentes dispositivos joguem juntos.
